# 配合《专业水报告漏洞模板.docx》食用更舒服
# 使用方式：
    py3 WaterExp.py -u http://www.target.com
    py3 WaterExp.py -f urls.txt

    扫描完毕复制漏洞名称
    打开模板文档
    ctrl+a打开微信截图，
    圈上对应响应头
    粘贴上文字
    粘贴到模板文档对应位置
    复制到新文档
    好了，可以出报告了
